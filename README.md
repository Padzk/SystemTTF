# SystemTTF
A TrueType version of the "System" bitmap font.

All glyphs are taken from all the vgasys.fon variations located in the Fonts folder from Windows 7/XP/10:\
 -fixedsys874.fon\
 -fixedsys1255.fon\
 -fixedsys1256.fon\
 -fixedsys1257.fon\
 -fixedsys1257.fon\
 -fixedsys.fon\
 -fixedsyse.fon\
 -fixedsysg.fon\
 -fixedsysr.fon\
 -fixedsyst.fon\

**Currently supports:**\
 • Turkish (vgasyst.fon)\
 • Greek (vgasysg.fon)\
 • Cyrillic (vgasysr.fon)\
 • Thai (vgas874.fon)\
 • Hebrew (vgasys1255.fon)\
 • Arabic (vgas1256.fon)\
 • Baldic (vgas1257.fon)\
 
WARNING: Arabic only displays in isolated forms
